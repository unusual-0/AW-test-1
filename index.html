Backend Setup Instructions
  1. Open the Google Sheet you want to write tree and expense logs to.
  2. Go to Extensions > Apps Script.
  3. Delete all code in the script editor.
  4. Copy and paste our specialized backend code.
  5. Click Deploy > New Deployment.
  6. Select Web App. Change access to "Anyone" (critical for offline sync).
  7. Authorize the permissions and copy the generated Web App URL.
  8. Paste the URL into the input field above.
